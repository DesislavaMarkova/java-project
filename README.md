# Music Playlist Manager

Конзолно приложение за управление на музикални плейлисти, разработено като курсов проект.

## Функционалности:
- Добавяне и управление на песни и жанрове.
- Създаване на плейлисти и изчисляване на времетраене.
- История на слушанията и справки за топ песни.
- Запис и зареждане на данни чрез сериализация (.bin файлове).

## Как се стартира:
Стартирайте `Main.java` и използвайте командите в терминала.
